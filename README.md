# LinkSwift Desktop

**网盘分享链接 → 一键转存 → 自动推送下载**

一个轻量、安全、高效的 Windows 桌面工具，帮你从夸克网盘分享链接中快速获取文件并推送到 RPC 下载器。

## ✨ 功能特性

### 🚀 核心功能

| 功能 | 说明 |
|------|------|
| **智能链接解析** | 粘贴夸克网盘分享链接，自动解析文件列表 |
| **可视化文件选择** | 树形展示文件结构，支持勾选文件/文件夹、全选/反选 |
| **一键转存** | 将选中文件转存到你的夸克网盘指定目录 |
| **直链获取** | 自动解析转存后文件的下载直链 |
| **RPC 推送** | 一键推送至 Aria2 / 比特彗星 / AB Download Manager 等下载器 |

### 🎯 用户体验优化

| 功能 | 说明 |
|------|------|
| **WebView 扫码登录** | 内嵌浏览器窗口，扫码即可登录，告别手动复制 Cookie |
| **Cookie 自动管理** | 登录凭证自动保存，过期前提醒，一键重新登录 |
| **首次运行向导** | 首次打开时交互式引导，配置 RPC 和登录账号 |
| **系统托盘** | 最小化到系统托盘，后台静默运行 |
| **快捷键支持** | `Ctrl+V` 粘贴链接、`Ctrl+A` 全选文件、`Enter` 开始转存 |
| **深色/浅色主题** | 跟随系统或手动切换，夜间使用更舒适 |
| **实时进度反馈** | 转存进度、下载速度、预估剩余时间一目了然 |
| **智能错误提示** | 错误信息一键复制，方便反馈问题 |

### 📦 功能扩展

| 功能 | 说明 |
|------|------|
| **批量任务队列** | 同时添加多个分享链接，按顺序自动处理 |
| **下载历史管理** | 查看已完成的下载任务，支持重新推送失败的链接 |
| **自动重试机制** | 转存或 RPC 推送失败时自动重试（最多 3 次） |
| **文件预览** | 转存前预览文件大小、格式等基本信息 |
| **代理支持** | 支持 HTTP/HTTPS 代理，适配特殊网络环境 |
| **多 RPC 服务器** | 配置多个 RPC 下载器，灵活切换 |

## 🚀 快速开始

### 系统要求

| 要求 | 最低配置 |
|------|---------|
| 操作系统 | Windows 10 (1903) 或更高版本 |
| 运行时 | WebView2（Win10/11 通常已内置） |
| 磁盘空间 | 约 20 MB |
| 网络 | 需要互联网连接 |

### 安装方式

#### 方式一：下载安装包（推荐）

1. 前往 [Releases](../../releases) 页面
2. 下载最新版本的 `.msi` 或 `.exe` 安装包
3. 双击运行，按提示完成安装

#### 方式二：从源码构建

> 需要安装 [Rust](https://rustup.rs/) 和 [Node.js](https://nodejs.org/)

```bash
# 克隆仓库
git clone https://github.com/your-username/LinkSwift-Desktop.git
cd LinkSwift-Desktop

# 安装前端依赖
npm install

# 开发模式运行
npm run tauri dev

# 构建安装包
npm run tauri build
```

> 💡 更多构建细节请参阅 [开发者文档](DEVELOPMENT.md)

## 📖 使用教程

### 第一步：首次登录

首次打开软件会自动弹出设置向导：

1. **登录夸克网盘**
   - 向导会打开一个内嵌浏览器窗口
   - 使用夸克 App 扫码登录（或使用账号密码）
   - 登录成功后窗口自动关闭，凭证自动保存

2. **配置 RPC 下载器**
   - 选择下载器类型（Aria2 / 比特彗星 / ABDM）
   - 填写 RPC 地址和密钥
   - 点击"测试连接"确认可用

3. **完成配置**
   - 保存后即可进入主界面

### 第二步：解析分享链接

1. 复制夸克网盘分享链接（如 `https://pan.quark.cn/s/abc123`）
2. 在主界面链接输入框中粘贴（`Ctrl+V`）
3. 如果链接有密码，会弹出密码输入框
4. 点击 **"解析链接"** 按钮
5. 等待解析完成，文件列表自动展示

### 第三步：选择文件

- **勾选文件**：点击文件前的复选框
- **全选/反选**：使用列表上方的快捷按钮
- **展开文件夹**：点击文件夹名称查看子文件
- **文件预览**：悬停文件可查看大小、格式等信息

### 第四步：转存并推送

1. 选择转存目标目录（默认为网盘根目录）
2. 选择 RPC 下载器
3. 点击 **"开始转存并推送下载"**
4. 软件会自动完成以下步骤：
   - 将选中文件转存到你的夸克网盘
   - 获取转存后文件的下载直链
   - 推送直链到 RPC 下载器
5. 在任务队列中查看进度和结果

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + V` | 粘贴链接并自动解析 |
| `Ctrl + A` | 全选文件 |
| `Ctrl + Enter` | 开始转存并推送 |
| `Ctrl + ,` | 打开设置 |
| `Ctrl + T` | 切换主题 |
| `Esc` | 关闭弹窗 / 取消操作 |

## ⚙️ 设置说明

### 账号管理

| 设置项 | 说明 |
|--------|------|
| **登录状态** | 显示当前夸克账号登录状态 |
| **重新登录** | Cookie 过期时点击重新扫码登录 |
| **自动刷新** | 开启后会在 Cookie 即将过期时自动提醒 |

### RPC 下载器配置

支持以下下载器：

| 下载器 | RPC 地址格式 | 备注 |
|--------|-------------|------|
| **Aria2** | `http://host:port/jsonrpc` | 最常用，支持 HTTP/HTTPS |
| **比特彗星** | `http://host:port` | 需开启 Web UI |
| **AB Download Manager** | `http://host:port/api` | 需开启远程 API |

> 💡 支持同时配置多个 RPC 服务器，可在推送时灵活切换。

### 代理设置

如果你需要通过代理访问夸克网盘：

1. 打开设置 → 网络设置
2. 填写代理地址（如 `http://127.0.0.1:7890`）
3. 如需认证，填写用户名和密码
4. 点击"测试连接"验证

### 主题设置

- **浅色模式**：适合白天使用
- **深色模式**：适合夜间使用
- **跟随系统**：自动切换

## ❓ 常见问题

### Q: 登录凭证多久过期？

A: 夸克网盘的登录凭证通常有效期为 **30 天左右**。软件会在凭证即将过期时提醒你重新登录，只需再次扫码即可。

### Q: 支持哪些网盘？

A: 当前版本支持 **夸克网盘**。后续计划支持阿里云盘、百度网盘、迅雷云盘等更多网盘。

### Q: 转存文件保存在网盘的哪个位置？

A: 默认保存在你的夸克网盘根目录。你可以在转存前选择目标文件夹，也可以在设置中修改默认转存目录。

### Q: RPC 推送失败怎么办？

A: 请检查以下几点：
1. RPC 下载器是否已启动
2. RPC 地址和端口是否正确
3. 防火墙是否阻止了连接
4. RPC 密钥是否匹配

软件会在推送失败后自动重试最多 3 次。

### Q: 支持哪些 RPC 下载器？

A: 目前支持 Aria2、比特彗星、AB Download Manager。只要下载器实现了标准的 JSON-RPC 接口，理论上都可以使用。

### Q: 软件会上传我的数据吗？

A: **不会**。所有数据（Cookie、配置、下载历史）均保存在本地，软件不会向任何第三方服务器上传用户数据。网络请求仅用于与夸克网盘和你的 RPC 下载器通信。

### Q: 分享链接有密码怎么办？

A: 粘贴链接后，软件会自动检测是否需要密码，并弹出密码输入框。

### Q: 可以同时处理多个链接吗？

A: 可以。你可以将多个分享链接添加到任务队列，软件会按顺序自动处理。

## 🛡️ 安全说明

| 安全措施 | 说明 |
|---------|------|
| **凭证加密** | 登录凭证使用 Windows DPAPI 加密存储，其他程序无法读取 |
| **本地存储** | 所有数据仅保存在本地 `%APPDATA%` 目录 |
| **零上传** | 不向任何第三方服务器发送用户数据 |
| **HTTPS** | 所有网络请求均使用 HTTPS 加密传输 |
| **日志脱敏** | 日志中的敏感信息（Cookie、Token）自动脱敏 |
| **开源透明** | 代码完全开源，接受社区审计 |
